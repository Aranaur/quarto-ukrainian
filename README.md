# Ukranian Quarto
Quarto localization into Ukranian

| File               | Language          |
| ------------------ | ----------------- |
| `_language-ua.yml`    | Ukranian |

## Usage

1. Download `_language-ua.yml` file and put it in root folder of Quarto project.

2. Add file name to YAML header

```yaml
---
title: "My Document"
language: _language-ua.yml
---
```
